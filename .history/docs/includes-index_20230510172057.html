Using Javascript to include one file inside another

You can reuse the same content multiple times across multiple pages, by way of Javascript.

Mixing unique content with reusable content

Some parts of the page might be unique to the page, and others might be used in more than one place.

You can use <div w3-include-html="...">...<div> and some Javascript to help with this.

Not all servers and security systems allow you to do this, though, so test on the server before relying too heavily on it.

You'll need to make sure links are relative to the server root so they'll work from different locations.

Full details are at https://www.w3schools.com/howto/howto_html_include.asp , or you can just view the source on this page.
